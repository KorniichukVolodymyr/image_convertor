## Assembly for image processing

### Installation

```bash
$ npm install
```

### Requirements

- node >= 14

***

### Supported formats

|  type  | minimization | conversion |    result    |
|:------:|:------------:|:----------:|:------------:|
| `jpg`  |      *       |     *      | `jpg + webp` |
| `jpeg` |      *       |     *      | `jpg + webp` |
| `png`  |      *       |     *      | `png + webp` |
| `webp` |              |            |              |
| `svg`  |      *       |            |    `svg`     |
| `gif`  |      *       |            |    `gif`     |
